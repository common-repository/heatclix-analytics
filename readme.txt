== Installation ==

= Deutsch: =

Installieren Sie Heatclix in einem einfachen WordPress-Blog
1. Laden Sie das komplette `heatclix`-Verzeichnis in Ihr `wp-content/plugins`-Verzeichnis hoch.
2. Aktivieren Sie das Plugin über das Menü "Plugins" in WordPress.
3. Öffnen Sie das neue 'Heatclix Plugin' und folgen Sie den Anweisungen, um Ihr Heatclix-Tracking zu konfigurieren.
4. Einstellungen speichern.

__________________________________________________________________________________________

= English: =

Install Heatclix on a simple WordPress blog
1. upload the complete `heatclix` directory to your `wp-content/plugins` directory.
2. activate the plugin from the "Plugins" menu in WordPress.
3. open the new 'Heatclix Plugin' and follow the instructions to configure your Heatclix tracking.
4. save settings.
